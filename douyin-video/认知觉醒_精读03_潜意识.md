## 📝 精读脚本

### 第一幕：潜意识的秘密（0-20秒）

```
第二章：潜意识

潜意识，是生命留给我们的彩蛋

人生是一场消除模糊的比赛
而潜意识，正是模糊的根源

为什么我们总是：
知道但做不到？
想要但得不到？

因为潜意识的力量
比意识大了3万倍以上
```

### 第二幕：感性的力量（20-45秒）

```
顶级成长者都在用"凭感觉"学习

什么叫凭感觉？

读到某段话时
突然心里一颤、头皮发麻、眼前一亮
那就是你的潜意识在告诉你：
这个重要，这个有用！

先大量读书
让兴趣、痛苦、需求触发潜意识
然后凭感觉找出最触动你的那一个
```

### 第三幕：如何利用潜意识（45-70秒）

```
三个方法利用潜意识：

第一，最佳策略是小人弹射
让理智和感性pk
哪个感受更强烈，哪个就赢

第二，总"界"的智慧
物无边，故似之
看一个人好不好
就看他让你难受还是舒服

第三，用同样的方式教孩子
想让孩子做什么
就让他在这件事上感受到
乐趣、痛苦和情绪
```

### 第四幕：记住这句话（70-85秒）

```
记住：
潜意识，就是你生命的彩蛋

拆开它
你就能打破人生的迷茫

《认知觉醒》精读第二期
下期预告：元认知——人类的终极能力

我是XX，关注我，一起觉醒
```
