## 📝 精读脚本

### 第一幕：专注的真相（0-20秒）

```
第五章：专注力

情绪和智慧的交叉地带

我们以为专注是把注意力
一直钉在一个地方

但真正的专注是：
身心合一

做A想A，做B想B
这就是深度沉浸
```

### 第二幕：为什么走神（20-45秒）

```
为什么我们总是走神？

因为走神的时候
大脑在后台偷偷休息

走神有两种：
1. 主动走神：主动休息，放松大脑
2. 被动走神：逃避困难，陷入焦虑

被动走神会消耗精力
让你越休息越累
```

### 第三幕：提升专注的方法（45-70秒）

```
一招提振你的注意力：

主动让感受回归

跑步时感受呼吸
吃饭时感受味道
读书时感受心流

身心合一
就是专注的最高境界

深度沉浸是进化双刃剑的安全剑柄
```

### 第四幕：记住（70-85秒）

```
专注不是死死盯住
而是身心合一

当你学会主动休息
主动感受
你就能掌控自己的注意力

《认知觉醒》精读第四期
下期预告：学习力——学习不是一味地努力

我是XX，关注我，一起觉醒
```
