## 📝 精读脚本

### 第一幕：人生五件套（0-20秒）

```
第八章：早冥读写跑

人生五件套
成本最低的成长之道

早起
冥想
阅读
写作
运动

这五件事
不需要天赋
不需要金钱
只需要坚持
```

### 第二幕：早起（20-40秒）

```
无闹钟、不参团、不打卡

早起为什么重要？

因为早起能给你一段
不受干扰的黄金时间

在这段时间里
你可以读书、写作、冥想
做真正重要的事
```

### 第三幕：冥想与阅读（40-60秒）

```
冥想：终有一天，你要解锁这条隐藏赛道

冥想不是玄学
而是大脑的健身操

阅读：如何让自己真正爱上阅读？

不是多读
而是读透一本胜读十本

让阅读改变你的认知
而不是填充你的谈资
```

### 第四幕：写作与运动（60-80秒）

```
写作：谢谢你，费曼先生

用最简单的话
解释最复杂的事

这叫费曼学习法
也是写作的本质

运动：灵魂想要走得远
身体必须在路上

运动不仅锻炼身体
更锻炼大脑
```

### 第五幕：结语（80-100秒）

```
一流的生活不是富有
而是觉知

觉知
就是知道自己想要什么
知道自己能做什么
知道自己该成为什么样的人

从今天开始
早睡早起、冥想、阅读、写作、运动

你就是那个觉醒的人
```

### 结尾（100-110秒）

```
《认知觉醒》精读系列
到这里就结束了

这本书告诉我们：
真正的成长
是认知的觉醒

我是XX
感谢观看
我们下本书见！
```
