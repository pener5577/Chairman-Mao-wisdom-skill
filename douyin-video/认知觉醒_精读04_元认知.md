- **画风推荐**：治愈动漫 / 清新日漫

## 📝 精读脚本

### 第一幕：什么是元认知（0-20秒）

```
第四章：元认知

元认知——人类的终极能力

什么是元认知？

就是当你举起锤子的时候
你突然意识到：
"我为什么要举锤子？"
"举锤子是最好的方式吗？"
"有没有更好的方式？"

这种"飞"起来看自己的能力
就是元认知
```

### 第二幕：成长慢的原因（20-45秒）

```
为什么你成长慢？

因为你不会"飞"

不会元认知的人
只能被自己的思维和行为带着走

别人骂你一句，你回骂一句
这就是没有元认知

而有元认知的人会想：
"他为什么骂我？"
"我应该怎么回应才最好？"
```

### 第三幕：如何获得元认知（45-70秒）

```
三个方法获得元认知：

第一，学习前人的智慧
读认知心理学、思维习惯等书籍
了解人类的思维陷阱

第二，时常反思
每天睡前问自己：
今天哪里做得不好？
哪里可以做得更好？

第三，冥想
专注呼吸，感受身体
你能"感受到自己在感受"
这就是元认知在增强
```

### 第四幕：成为思维舵手（70-85秒）

```
我们生而为人
就是为了成为思维舵手

元认知强的人
能在任何时刻
跳出自己看自己

他能选择情绪
而不是被情绪选择

《认知觉醒》精读第四期
下期预告：专注力——情绪和智慧的交叉地带

我是XX，关注我，一起觉醒
```
