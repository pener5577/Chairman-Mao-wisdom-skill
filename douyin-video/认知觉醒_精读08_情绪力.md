# 《认知觉醒》精读系列 - 第八期

## 视频信息
- **时长**：60-90秒（精读版）
- **主题**：第七章：情绪力——情绪是多角度看问题的智慧
- **画风推荐**：治愈动漫 / 清新日漫

---

## 📝 精读脚本

### 第一幕：情绪的真相（0-20秒）

```
第七章：情绪力

情绪是多角度看问题的智慧

为什么你总是焦虑？
因为视角太单一

为什么你总是愤怒？
因为你只看到一面
```

### 第二幕：心智带宽（20-45秒）

```
唯有富足，方能解忧

什么是心智带宽？
就是大脑处理信息的空间

当你的心智带宽被占满时
你就无法思考
只能本能反应

所以越穷的人越容易短视
越忙的人越容易焦虑
```

### 第三幕：多角度看问题（45-70秒）

```
你的坏情绪
源于视角单一

为什么有人能从失败中看到机会？
因为他有多角度看问题的能力

怎么练？
第一，看到成功，也要看到失败
第二，看到局部，也要看到全局
第三，看到现在，也要看到未来
```

### 第四幕：游戏心态（70-85秒）

```
幸福的人，总是在做另外一件事

把"我必须成功"
变成"我先试试看"

把"这件事太难了"
变成"这件事有趣在哪"

游戏心态，不是玩世不恭
而是放下执念，轻装上阵
```

### 结尾（85-100秒）

```
情绪不是敌人
情绪是指南针

学会解读情绪
你就能多角度看问题

《认知觉醒》精读第七期
下期预告：早冥读写跑——人生五件套

我是XX，关注我，一起觉醒
```
