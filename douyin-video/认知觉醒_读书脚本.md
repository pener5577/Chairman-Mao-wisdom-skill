# 《认知觉醒》读书视频脚本

## 视频信息
- **时长**：45-60秒
- **风格**：干货、励志
- **主题**：如何开启自我改变的原动力

---

## 📝 分镜脚本

### 第一幕（开场 - 痛点）0-8秒
```
你是否也这样：
每天忙忙碌碌，却感觉没有成长
想努力，却总是敌不过惰性
想精进，面前却总是弯路不断

读了很多书，都忘了
付出很多努力，都白费了
```

### 第二幕（觉醒时刻）8-18秒
```
这就是"醒着的睡着的人"
作者周岭36岁才觉醒

直到身边两位好友命运突变
他才意识到：
如果意外发生在自己身上
我还有什么，会什么？
```

### 第三幕（核心观点）18-35秒
```
认知觉醒的关键：

第一，大脑认知
我们一切问题的根源，在于大脑
了解它，才能控制它

第二，潜意识
人生是一场消除模糊的比赛
用感性察觉，用理性思考

第三，元认知
成长慢，是因为你不会"飞"
能随时反思自己，才是顶级能力
```

### 第四幕（方法论）35-50秒
```
如何真正改变？

学习力——在舒适区边缘刻意练习
行动力——懂得百遍不如行动一遍
情绪力——多角度看问题

早睡早起、冥想、阅读、写作、运动
人生五件套，成本最低的成长之道
```

### 第五幕（结尾）50-60秒
```
《认知觉醒》
开启自我改变的原动力

我是XX，关注我，下期分享更多好书
```
