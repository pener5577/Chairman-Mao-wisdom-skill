# 《认知觉醒》精读系列 - 第七期

## 视频信息
- **时长**：60-90秒（精读版）
- **主题**：第六章：行动力——没有行动世界只是个概念
- **画风推荐**：治愈动漫 / 清新日漫

## 📝 精读脚本

### 第一幕：行动力的本质（0-20秒）

```
第六章：行动力

没有行动
世界只是个概念

为什么我们懂得很多道理
却依然过不好这一生？

因为我们只在"脑内"学习
没有在"体外"行动
```

### 第二幕：清晰是行动的前提（20-45秒）

```
一个观念重构你的行动力：

行动力，源于清晰

你对要做的事越清晰
行动力就越强

为什么你总是拖延？
因为目标太模糊
因为你不确定能不能做到

把"我要读书"
变成"每天晚上8点读10页"
行动力就会爆发
```

### 第三幕：做一个"傻瓜"（45-70秒）

```
这个世界会奖励那些
不计得失的"傻瓜"

想太多会阻碍行动
先做再说

道理都懂，就是不做
怎么破解？

答案只有一个：
不是学不会，是不去做
不是不知道，是没做到
```

### 第四幕：如何开始（70-85秒）

```
立刻行动的方法：

不要等到准备好了才行动
而是在行动中准备

不要等到状态好了才行动
而是用行动带动状态

做一个行动上的"傻瓜"
比做一个思想上的"巨人"更有用

《认知觉醒》精读第六期
下期预告：情绪力——情绪是多角度看问题的智慧

我是XX，关注我，一起觉醒
```
