## 📝 精读脚本

### 第一幕：大脑的构成（0-25秒）

```
第一章：大脑——一切问题的起源

我们的大脑分为三层：

本能脑——爬行动物就有的
主管生存本能

情绪脑——哺乳动物就有的
主管情绪、感受

理智脑——人类独有的
主管认知、思考、规划

问题在于：
理智脑太年轻了！
大约只发育了250万年
而本能脑已经2亿年了
```

### 第二幕：焦虑的根源（25-55秒）

```
为什么我们总是焦虑？

因为本能脑和情绪脑
太！强！大！了！

它们追求：
简单、轻松、舒适、确定

所以我们天生：
避难趋易
急于求成

这就是焦虑的根源！

不是我们不够努力
而是我们的本能太强大
```

### 第三幕：什么是真正的耐心（55-85秒）

```
得耐心者得天下

假耐心：
强迫自己忍受痛苦
逼自己死撑

真耐心：
能看到长远价值
能接受暂时的缓慢

真正的耐心
是知道自己的努力
终将开花结果
```

### 第四幕：如何增强耐心（85-110秒）

```
三个方法增强耐心：

第一，延迟满足
先做重要的事
再做快乐的事

第二，拥抱困难
困难=成长
逃避=退步

第三，看到复利
任何技能的学习
都有复利效应
前期慢，后期快
```

### 结尾（110-120秒）

```
这一章告诉我们：

大脑是一切问题的起源
焦虑是本能太强
耐心是看到长远

下期预告：潜意识——生命留给我们的彩蛋

我是XX，关注我，一起觉醒
```
