## 🎨 什么是文生图？

通过文字描述生成图像的 AI 技术
- 输入：文字提示词
- 输出：精美的图像

## 🔥 热门工具推荐

**特点：**
- 艺术性强，细节丰富
- 适合创意设计和概念图
- Discord 平台使用

**入门提示：**
```
/imagine prompt: 一个未来城市，霓虹灯，赛博朋克风格
```

## 💡 提示词技巧

**基础公式：**
```
[主体] + [风格] + [光影] + [构图] + [细节]
```

**示例：**
```
一个可爱的机器人，
赛博朋克风格，
霓虹灯光，
黄金分割构图，
4K 超高清
```
