# 网络安全工具介绍
## 专业安全人员的必备工具

---

### 1. Nmap - 网络扫描器
**用途：** 发现网络中的主机和服务
**合法用途：** 网络资产管理、安全审计

```bash
nmap -sV 192.168.1.1  # 扫描服务版本
```

---

### 2. Nikto - Web 服务器扫描器
**用途：** 检测 Web 服务器漏洞
**合法用途：** Web 应用安全测试

---

### 3. Wireshark - 网络分析工具
**用途：** 捕获和分析网络数据包
**合法用途：** 网络故障排查、安全分析

---

### 4. Burp Suite - Web 安全测试
**用途：** 拦截和修改 HTTP 请求
**合法用途：** 渗透测试、安全研究

---

### ⚠️ 重要提醒
- 仅在对拥有授权的目标使用
- 学习是为了更好的安全防护
- 遵守法律法规
