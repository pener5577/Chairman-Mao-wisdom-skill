## 一、豆包（Doubao）高效使用法

### 1.1 核心心法：三轮对话法

| 阶段 | 目标 | 提问示例 |
|------|------|----------|
| 第一轮 | 搭建框架 | "帮我规划一个新品发布会" |
| 第二轮 | 验证数据 | "你提到预算2万，这个标准是什么？" |
| 第三轮 | 检查盲点 | "作为新手，还有什么容易忽略的风险？" |

### 1.2 优化简历提示词模板

```
基于以下简历和[岗位名称]，帮我精修：
1. 个人信息只留姓名、电话、邮箱
2. 用STAR法则重构经历，每个加量化数据
3. 融入岗位关键词
4. 给出排版方案
原始简历：[粘贴]
```

### 1.3 演讲稿提示词

```
为[行业]写[场景]发言稿：
- 时长：[5分钟]
- 风格：[正式/轻松/专业]
- 核心信息：[列出]
- 目标受众：[描述]
```

## 二、即梦（Jimeng/Dreamina）图像视频生成

### 2.1 图像生成公式

**主体 + 环境 + 风格**

| 要素 | 示例 |
|------|------|
| 主体 | 年轻女孩、小麦色皮肤、金色头发 |
| 环境 | 金字塔、古埃及建筑 |
| 风格 | 软光、侧光、电影感 |

### 2.2 人像写真操作步骤

1. 打开即梦AI → 图片生成
2. 上传清晰正脸照片
3. 选择"参考图"模式 → 人像写真
4. 输入提示词（主体+环境+风格）
5. 选择模型 Image 3.1
6. 设置比例：3:2(横版) 或 3:4(竖版)
7. 点击生成

### 2.3 视频生成操作步骤

1. 打开即梦AI → 视频生成
2. 上传静态图片或首帧/尾帧图片
3. 输入动作描述：如"人物抬头眺望远方，目光柔和"
4. 选择模型：Video S2.0
5. 设置比例和时长（5秒）
6. 点击生成

### 2.4 老照片动画

- 上传老照片
- 选择S2.0模型
- 可选：输入动作提示词，或让AI自动生成
- 自动让人像眨眼、微笑，背景动态化

### 2.5 首帧尾帧视频生成

1. 上传首帧图片（开头画面）
2. 上传尾帧图片（结尾画面）
3. 输入动作描述：如"花朵绽放，镜头缓慢推进"
4. 选择Video 3.0模型
5. AI自动生成中间过渡帧

## 三、剪映（Jianying/CapCut）AI功能

### 3.1 AI带货文案生成

**操作步骤：**
1. 打开剪映专业版 → AI功能
2. 选择"生成带货文案"
3. 设置：视频比例(9:16)、时长(15-30秒)
4. 输入产品信息：
   - 产品名称
   - 卖点
   - 目标人群
   - 促销信息
5. 点击"生成文案"
6. 生成5个脚本，选择想要的
7. 点击"生成视频"

### 3.2 AI智能剪辑

**操作步骤：**
1. 导入视频素材
2. 选择脚本（AI生成或手动选择）
3. 点击"AI智能剪辑"
4. 自动完成：配乐、特效、字幕、配音
5. 预览并导出

### 3.3 数字人制作

**位置：** 剪映 → AI功能 → 数字人

**功能：**
- 选择数字人形象
- 输入文案
- AI自动生成口播视频

## 四、变现内容制作流程

### 4.1 抖音爆款内容公式

| 阶段 | 时长 | 内容 |
|------|------|------|
| 黄金钩子 | 0-3秒 | 引发好奇/痛点/悬念 |
| 干货输出 | 15-45秒 | 核心价值内容 |
| 引导互动 | 3-5秒 | 点赞/关注/评论 |

### 4.2 短视频制作SOP

**第一步：选题**
- 用豆包搜索热点话题
- 选择领域：职场、情感、知识、生活

**第二步：写脚本**
- 用豆包生成脚本
- 用剪映AI优化文案

**第三步：生成素材**
- 用即梦生成图片/视频
- 或用剪映AI生成视频

**第四步：剪辑**
- 剪映：添加字幕、BGM、特效
- 用AI配音/数字人

**第五步：发布**
- 检查封面标题
- 添加话题标签
- 发布到抖音

## 五、变现模式总结

### 5.1 内容流量变现

| 模式 | 操作 | 收入预期 |
|------|------|----------|
| 流量分成 | 发视频→平台给分成 | 1万播放≈10元 |

| 服务 | 操作 | 定价 |

| 阶段 | 粉丝量 | 变现方式 |
|------|--------|----------|
| 起步期 | 0-1000 | 赚流量费 |
| 成长期 | 1000-5000 | 置换/广告 |
| 成熟期 | 5000+ | 广告+带货+课程 |
